1. Introduccion a Linux

Que es Linux

Linux es un kernel y un ecosistema de distribuciones que permiten construir sistemas seguros, estables y abiertos.

Distribuciones populares

  • Ubuntu y Linux Mint para escritorio
  • Debian y Fedora para entornos estables
  • Arch y Manjaro para usuarios avanzados

Filosofia Unix

Herramientas pequenas, combinables y con salida de texto para resolver tareas complejas usando pipelines.

Por que aprender Linux

Es clave en servidores, DevOps, cloud, ciberseguridad y desarrollo moderno.

3. Comandos basicos

Haz clic en Ejecutar para ver la simulacion en la terminal.

pwd

Imprime el directorio actual.

$ pwd

ls

Lista archivos y carpetas.

$ ls

cd

Cambia de directorio.

$ cd proyectos

mkdir

Crea directorios.

$ mkdir nueva_carpeta

rm

Elimina archivos o carpetas.

$ rm archivo.txt

cp

Copia archivos y carpetas.

$ cp archivo.txt copia.txt

mv

Mueve o renombra archivos.

$ mv viejo.txt nuevo.txt

touch

Crea archivos vacios o actualiza fecha.

$ touch notas.txt

cat

Muestra contenido de archivos.

$ cat archivo.txt

nano

Editor de texto en terminal.

$ nano archivo.txt

clear

Limpia la terminal.

$ clear

history

Muestra el historial de comandos.

$ history

4. Sistema de archivos

/

Raiz del sistema, todo parte de aqui.

/home

Carpetas de usuarios y documentos personales.

/etc

Configuraciones del sistema y servicios.

/var

Datos variables como logs y cache.

/usr

Aplicaciones y librerias compartidas.

/tmp

Archivos temporales, se limpia con el tiempo.

5. Permisos

rwx

r: lectura, w: escritura, x: ejecucion.

chmod u+rwx archivo.sh

755 y 644

755 = propietario rwx, grupo r-x, otros r-x. 644 = propietario rw-, grupo r--, otros r--.

chmod 755 script.sh

chmod

Cambia permisos de archivos.

chmod 644 archivo.txt

chown

Cambia propietario y grupo.

chown usuario:grupo archivo.txt

6. Procesos

ps

Lista procesos activos.

ps aux

top

Monitor en tiempo real.

top

htop

Monitor mejorado (instalar).

htop

kill

Finaliza procesos por PID.

kill -9 1234

7. Redes

ip

Gestiona interfaces y rutas.

ip a

ping

Prueba conectividad.

ping -c 4 8.8.8.8

curl

Descarga y prueba APIs.

curl https://example.com

wget

Descarga archivos.

wget https://example.com/archivo.zip

netstat

Conexiones y puertos.

netstat -tulpn

8. Usuarios

useradd

Crea usuarios.

sudo useradd -m dev

passwd

Asigna contrasena.

sudo passwd dev

su

Cambiar de usuario.

su - dev

sudo

Eleva privilegios de forma segura.

sudo apt update
