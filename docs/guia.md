# Guia interactiva de Linux desde cero

## Objetivo
Aprender los fundamentos de Linux con ejemplos reales y una terminal simulada.

## Como usar el sitio
1. Abre `index.html` en tu navegador.
2. Usa el buscador para filtrar comandos.
3. Presiona "Ejecutar" para ver la simulacion.

## Practicas recomendadas
- Ejecuta los comandos en un entorno seguro como WSL o una VM.
- Repite cada comando y modifica sus argumentos.
- Completa el script `bash/scripts.sh` y observa los cambios.

## Ejercicios
- Crea una carpeta llamada `laboratorio` en tu home.
- Crea un archivo `notas.txt` y escribe 3 lineas.
- Cambia los permisos a 644 y verifica con `ls -l`.

## Resultado esperado
Al terminar deberias poder navegar, listar, crear y administrar archivos en Linux.
